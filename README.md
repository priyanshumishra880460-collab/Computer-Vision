# 🧠 Azure Computer Vision Web App

✨ A modern web application that uses **Microsoft Azure Computer Vision API** to analyze images and generate AI-based captions.

---

## 🚀 Features

✔ 🔑 Enter Azure API Key & Endpoint  
✔ 🌐 Analyze images using URL  
✔ 📤 Upload and analyze local images  
✔ 🖼 Live image preview  
✔ 🤖 AI-generated captions  

---

## 🛠 Tech Stack

- 🌐 HTML  
- 🎨 Tailwind CSS  
- ⚡ JavaScript (Fetch API)  
- ☁️ Azure Computer Vision API

## ⚙️ How to Use

1️⃣ Open `index.html` in your browser  
2️⃣ Enter your Azure API Key & Endpoint  
3️⃣ Choose one option:  
   - Paste Image URL  
   - Upload Image  
4️⃣ Click **Analyze**  
5️⃣ View AI-generated result
